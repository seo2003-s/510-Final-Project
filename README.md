# 510-Final-Project

# Project Description
The goal of this project is to analyze the different datasets related to cardiovascular health: heart disease, heart failure, and general cardiovascular disease. 

## How to Run

### 1. Clone the repository
```bash
git clone https://github.com/seo2003-s/510-Final-Project.git
cd 510-Final-Project
```

### 2. Install dependencies
```bash
pip install -r requirements.txt
```
